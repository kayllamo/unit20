Part 1

  • What is HTTP?
      • HTTP is hypertext transfer protocol. Method for encoding and transporting information from browser to server.
  • What is a URL?
      • A URL is an uniform resource locator or an address for a website.
  • What is DNS?
      • DNS is a domain name system that turns a domain name into IP addresses.
  • What is a query string?
      • Part of the URL, a query string assigns values to the specified parameters like directing users to a certain page.
  • what are two HTTP verbs and how are they different?
      • GET and POST. GET is used to retrieve information while POST is used to create a new item or resource.
  • What is an HTTP request?
      • A request is made by the browser/client to a server in exchanging information.
  • What is an HTTP response?
      • A reponse is the server giving information to the client or web browser.
  • What is an HTTP header? Give examples of request and response headers you have seen.
      • An HTTP header is the value pair in the request and response messages.
